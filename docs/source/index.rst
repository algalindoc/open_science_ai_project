Bienvenido a la documentación del **Open Science AI Project**.  
Este proyecto utiliza **Grobid** y herramientas de análisis de datos en **Python** para extraer información de documentos científicos en formato PDF.  
Con **Docker**, se garantiza una ejecución reproducible y sin complicaciones.

Para comenzar a utilizar este proyecto, consulta la sección :doc:`readme`.

Si deseas conocer más sobre las decisiones técnicas y la validación de datos, revisa el :doc:`rationale`.
